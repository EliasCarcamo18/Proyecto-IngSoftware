# EcoRoute Logistic AI - Prototipo funcional completo

Este proyecto es una página web funcional para Transportes Sur-Austral.

## Archivos principales

- `index.html`: estructura de la página.
- `styles.css`: diseño visual, colores y responsive.
- `script.js`: lógica funcional, base local, validaciones, CRUD y navegación.

## Cómo ejecutar

1. Descarga el ZIP.
2. Descomprímelo.
3. Abre `index.html` en Google Chrome o Microsoft Edge.

## Funciones incluidas

- Registro, edición, eliminación y listado de despachos.
- Consulta de envío por número de guía.
- Gestión de usuarios y trabajadores.
- Gestión de vehículos.
- Gestión y cálculo de rutas sugeridas.
- Gestión de alertas climáticas.
- Notificaciones visuales automáticas.
- Base de datos local con IndexedDB.
- Exportación e importación de respaldo JSON.
- Datos persistentes en el navegador.

## Observación

El prototipo usa datos locales simulados. No requiere servidor, MySQL ni conexión real a GPS o clima.
